# 酒店管理系统

## 一、概述

酒店管理系统，为大三上学期**JavaWeb课程大作业**。项目配置项为：

* 服务器：apache-tomcat-9.0.0.M26

* 数据库：Mysql 5.7.18-1

* 编辑器：IntelliJ IDEA 2017.0.5
